## Binge Box GPT

### Features
- Authentications
- Browse Page
- Suggestion of Movies from GPT API
- Video Details Page
- Movie Trailer on Background
- Play Video on hover

- Login/Sign Up
    - Sign In /Sign up Form

- Browse (after authentication)
    - Header
    - Main Movie
        - Tailer in Background
        - Title & Description
        - MovieSuggestions
        - MovieLists * N
- NetflixGPT
    - Search Bar
    - Movie Suggestions
